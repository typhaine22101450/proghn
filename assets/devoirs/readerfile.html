Bienvenue dans le centre d'aide !
Nous vous expliquons ici à quoi servent les différents boutons.

Segmentation : cliquez dessus pour segmenter votre texte en mots. Vous connaitrez ainsi toutes les formes uniques présentes, leur nombre d'occurences, triées par leur nombre de caractère.
Vous pouvez utiliser les délimiteurs par défaut ou les personnaliser à votre guise.

Cooccurrents/fréquence : cliquez dessus pour visualiser sous forme de tableau les cooccurrences du mot cible (à inscrire dans "pôle"). Vous pouvez également entrer une "longueur" qui vous permettra de donner une distance maximale entre votre mot cible et ses cooccurrents.
Le tableau vous précisera également la fréquence gauche et droite des cooccurrents ainsi que la cofréquence.

Visualisation Cooccurrents : cliquez dessus pour afficher les mêmes résultats que pour "Cooccurrents/fréquences" mais sous forme d'un graphique en barre.
Barre jaune : Cofréquence
Barre orange : Fréquence gauche
Barre rouge : Fréquence droite
